Structure Factor, RDF Analysis
structureFactor

Post-processing code for RDF, in-plane RDF, structure factor

This C++ program reads in lammps trajectory files or XYZ files to calculate RDF, in-plane RDF and the structure factor.

Compilation

To generate the executable runme use:

make

An executable runme will be formed in the base directory.

Running

./runme
