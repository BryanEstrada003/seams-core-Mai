Detailed Description

Class for Structure Factor Calculation. This class creates an object for the structure factor, and the output can be printed to a file.

For example, if you want to find the in-plane RDF in the XY plane, you should use the initRDFxy() function, followed by either the accumulateRDFxy() or singleRDFxy(), depending on whether you want to average over several frames or not. If you use the accumulateRDFxy function, you must also use the normalizeRDF2D() function. Don't call call normalizeRDF2D() when using singleRDFxy(), since this function is called by singleRDFxy() itself. The printRDF2D() function prints the radial values and corresponding RDF values to a file called rdf2D.dat in the output folder.

The structure factor can be determined in two ways: directly from the coordinates or as a Fourier transform of the pair correlation function .
