\subsection{Detailed Description}
Class for 3D R\+DF. This class creates an object for the 3D-\/\+R\+DF, and the output of the R\+DF can be printed to a file. 

Use \mbox{\hyperlink{classCAnalysis_ac20cd821313a1e36ab0f63e93d6e4149}{init\+R\+D\+F3\+D()}} to create an object for the 3D R\+DF. Depending on the number of frames, use \mbox{\hyperlink{classCAnalysis_ab3566f13bda907427f28c82706ac3aec}{single\+R\+D\+F3\+D()}} or \mbox{\hyperlink{classCAnalysis_a5d6f09a6fd0f9756e69ace8f595756b6}{accumulate\+R\+D\+F3\+D()}} . In case you have used the accumulate function, you will have to use the \mbox{\hyperlink{classCAnalysis_a1716c9da84e9d41cbb32b97f7708e9f7}{normalize\+R\+D\+F3\+D()}} to normalize the R\+DF. Finally, print the output to an output file, which is called rdf3\+D.\+txt by default using the print function
